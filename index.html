Sistema de Gestión y Venta de Boletas

Proyecto - Programación 2

Descripción del Proyecto

Sistema desarrollado en Java con Maven que permite la gestión de eventos, reserva de boletas, registro de pagos y consulta de estados.

Requerimientos Principales

  • Creación de eventos con 3 zonas (A, B y C).
  • Reserva de máximo 10 boletas por usuario.
  • Validación de mayoría de edad (18+).
  • Reservas válidas por 24 horas.
  • Estados: Reservada, Pagada, Cancelada.

Tecnologías

  • Java
  • Maven
  • Git
  • GitHub

Control de Versiones

Ramas utilizadas:

  • master (versión estable)
  • develop (desarrollo)
  • evento (feature específica)
